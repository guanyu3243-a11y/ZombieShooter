# Zombie Shooter

## Game Overview
Zombie Shooter is a 3D survival shooting game developed using Unity.  
The player must survive against continuously attacking zombies while managing health.

---


## Features
- 3D player movement and shooting system
- Zombie AI chasing behaviour
- Zombie attack animation
- Health and damage system
- Score system
- Pause Menu (ESC)
- Settings Menu (Volume & Sensitivity)

---

## Controls
| Key | Action |
|-----|-------|
| W A S D | Move |
| Mouse | Aim |
| Press"J" | Shoot |
| ESC | Pause Menu |

---

## Game Mechanics Implemented
- A simple Enemy attack animation system
- Enemy AI behaviour
- Two Scenes-Game and Menu
- Game state management
- Player health and score system

---

## Built With
- Unity 2022.3
- C#
- TextMeshPro
- Unity Animator System

---

## How to Run
1. Open project using Unity Hub
2. Load the **Main Menu** scene
3. Press Play in Unity Editor
